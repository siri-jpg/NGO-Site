# InAmigos Foundation Awareness Website

A modern and responsive NGO awareness website developed for **InAmigos Foundation** using **HTML5** and **CSS3**.

## Features

- Responsive design
- Modern hero section
- About InAmigos Foundation
- NGO projects showcase
- Social impact statistics
- Recent events section
- Contact information
- Call-to-action for volunteering
- Clean and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome

## Purpose

This project was developed as part of an internship assignment to promote awareness about InAmigos Foundation, its initiatives, and its contribution to social welfare across India.
